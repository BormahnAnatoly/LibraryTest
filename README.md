# Registry Book
### Task
Делаем простое приложение "Реестр книг".
В главной форме в DataGrid отображаем текущий список книг (изначально пустой).
  - Колонки:
  - Инв. №
  - Название
  - Автор(ы)
  - Год издания
  - Дата поступления

 ##  Добавить четыре кнопки:
 
Добавить - в отдельном окне вводятся свойства новой книги, после OK добавляется новая книга в конец списка, Cancel - отмена
Редактировать - в отдельном окне редактируются свойства текущей книги, после OK вносятся изменения, Cancel - отмена
Удалить - после подтверждения удаляем книгу из списка
Сохранить в файл - сохраняем в файл текущий список книг

При старте ищем файл, если файла нет, то начинаем с пустого списка, если есть вычитываем все книги из этого списка..
